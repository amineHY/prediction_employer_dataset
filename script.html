Projet data science¶

Objectifs:

  • clustering non supervis afin d'identifier 2 groupes de profils techniques distinctes
  • prdiction des profils dont le mtier n'est pas labellis

Données¶

data.csv contient 6 variables :

- 'Entreprise' correspond  une liste d'entreprises fictive
- 'Metier' correspond au mtier parmi data scientist, lead data scientist, data engineer et data architecte
- 'Technologies' correspond aux comptences matrises par le profil
- 'Diplome' correspond  son niveau scolaire (Bac, Master, PhD,...)
- 'Experience' correspond au nombre d'annes d'expriences
- 'Ville' correspond au lieu de travail

def add_percentge(ax, x=0.5, y=0.25, direction='horizontal'):    
    totals=[]
    if direction=='horizontal':
        for i in ax.patches:
            totals.append(i.get_width())
        total = np.sum(totals)
        for i in ax.patches:
            str_percentage = str(round((i.get_width()/total)*100,2))+'%'
            ax.text(i.get_width()+x, i.get_y()+y, str_percentage, weight='bold', color='black')
            
    elif direction=='vertical':
        for i in ax.patches:
            totals.append(i.get_height())
        total = np.sum(totals)
        for i in ax.patches:
            str_percentage = str(round((i.get_height()/total)*100,2))+'%'
            ax.text(i.get_x()+x, i.get_height()+y, str_percentage,  weight='bold', color='black')

labels_ = kmeans.labels_


# Evaluation de l'erreur de clustering

NMSE = lambda a,b: 100 * ((a-b)**2).sum() / ((a)**2).sum()
